# Boids
Experiment using [PIXI][pixi] and Craig Reynolds' Boids.

![Boids](http://puga.com.br/play/boids/images/boids.png)

##### [Live demo][demo]

Make sure you have [Yo][yo], [Bower][bower] and [Gulp][gulp] installed.

	$  npm install -g bower yo gulp

How to run:

	$ npm install
	$ bower install
	$ gulp serve
